
# Example Library Starter

Included:
- Rollup
- Babel
- Sass (SCSS)
    - PostCSS, Autoprefixer
- nanocomponent
- nanohtml
- ramda
- core-js
- Support for IE11

## Getting Started
```bash
npx degit https://github.com/bradyhullopeter/rollup-lib-babel-starter.git#main my-lib
```

```bash
cd my-lib
```

```bash
npm install
```

```bash
npm run dev
```
