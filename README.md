
# Example Library Starter

Included:
- Rollup
- Babel
- Sass (SCSS)
    - PostCSS, Autoprefixer
- nanocomponent
- nanohtml
- ramda
- core-js
- Support for IE11

## Getting Started
```bash
git clone https://github.com/bradyhullopeter/rollup-lib-babel-starter.git mylib
```

```bash
cd mylib
```

```bash
npm install
```

```bash
npm run dev
```
